# Arquivo de descrição do problema
#
# Sintaxe:
#
# [elemento] [opcao1] [valor1] [opcao2] [valor2] ... [opcaon] [valorn]
#
# Elementos:
#
#  - viga: Características da viga
#
#          Opções: tipo [barra/cilindro/oco]               -> Tipo de viga
#                  modulo [elastico/cisalhamento] [numero] -> E/G (Pa)
#
#          - Para tipo barra:
#                  altura [numero]      -> Altura da viga (eixo y) (m)
#                  comprimento [numero] -> Comprimento da viga (eixo x) (m)
#                  largura [numero]     -> Largura da viga (eixo z) (m)
#
#          - Para tipo cilindro:
#                  raio [numero] -> Raio da viga (m)
#
#          - Para tipo oco:
#                  raio externo [numero] -> Raio externo da viga (m)
#                  raio interno [numero] -> Raio interno da viga (m)
#
#          Exemplos: viga tipo cilindro raio 3 modulo elastico 60000000000
#                    viga tipo barra altura 0.41 largura 1 comprimento 5
#
#  - forca: Força aplicada na viga
#
#          Opções: valor [numero]   -> Módulo da força, com sinal indicando o sentido* (N)
#                  vertical         -> Indica que a força é aplicada na direção vertical
#                  horizontal       -> Indica que a força é aplicada na direção horizontal
#                  posicao [numero] -> Posição da força no eixo x da viga (m)
#                  angulo [numero]  -> Ângulo com o vetor apontando no sentido horizontal positivo* (graus)
#
#          Exemplos: forca valor 10 horizontal posicao 2
#                    forca angulo 45 posicao 15 valor 12.1346
#
#  - apoio: Apoio colocado na viga
#
#          Opções: tipo [fixo/pino/rolete]   -> Tipo de apoio
#                  posicao [numero]          -> Posição do apoio no eixo x da viga (m)
#
#          Exemplos: apoio posicao 2 tipo rolete
#                    apoio tipo fixo
#
#  - momento: Momento aplicado no eixo z da viga
#
#          Opções: valor [numero]   -> Valor do momento (Nm)
#                  posicao [numero] -> Posição do momento no eixo x da viga (m)
#
#          Exemplos: momento valor 9.31
#                    momento posicao 61.413 valor 314
#
#  - torque: Torque aplicado na viga
#
#          Opções: valor [numero]   -> Valor do torque (Nm)
#                  posicao [numero] -> Posição do torque no eixo x da viga (m)
#
#          Exemplos: torque posicao 0 valor 763.12
#                    torque valor 1
#
#  - carregamento: Carregamento distribuido aplicado entre dois pontos da viga
#
#          Opções: coeficientes [numero] ... [numero]   -> Coeficientes do polinômio que descreve o carregamento, em ordem decrescente de grau (N/m)
#                  de [numero] ate [numero]             -> Posições inicial e final do carregamento no eixo x da viga (m)
#
#          Exemplos: carregamento de 0 ate 2 coeficientes 1 6 4 12 6 75 123
#                    carregamento coeficientes 9 9 3 12 87 24 8 1 8 0 2 de 1 ate 5
#
# * Para elementos com direção e sentido, são considerados positivos os sentidos para cima e para a direita, e negativos os sentidos para baixo e para a esquerda
